When we use our computer to type in characters and other commands from our keyboard, the computer doesn't see the commands as "A", "B", "C", but instead as numbers (KeyCodes), which then represents a character based which encoding is being used. With this in mind I decided to change the text input through manipulating and using each characters corresponding KeyCode, I did this in two different ways:
KeyCode Shifting - takes the keycode for every character and shifts it a set amount in one directions, by adding a number to the KeyCode.
KeyCode ordering - sums up the keycode for each word, and sorts all the words according to their keycode sum, in descending order.
